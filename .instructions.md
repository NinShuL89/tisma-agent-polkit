# Guía de Desarrollo - Tisma Polkit Agent

Este proyecto es un **PolicyKit Authentication Agent** moderno para Arch Linux escrito en Rust con interfaz gráfica en Vala + GTK4.

## Estructura del Proyecto

### Backend (Rust)
- `src/main.rs` - Punto de entrada del programa
- `src/polkit.rs` - Implementación del agente PolicyKit (D-Bus)
- `src/ui/mod.rs` - Módulo de interfaz con la UI de Vala

### Frontend (Vala + GTK4)
- `vala/polkit_dialog.vala` - Diálogo de autenticación visual

### Configuración del Sistema
- `dbus/org.tisma.PolkitAgent.xml` - Interfaz D-Bus
- `systemd/tisma-agent-polkit.service` - Servicio systemd
- `policies/org.tisma.policy` - Políticas de autorización
- `build.rs` - Script de compilación (compila Vala)

## Instalación de Dependencias (Arch)

```bash
sudo pacman -S rustup cargo gtk4 vala libadwaita polkit dbus

# Para compilación de Vala
sudo pacman -S vala-devel
```

## Compilación

```bash
# Debug
cargo build

# Release (recomendado)
cargo build --release
```

## Notas de Desarrollo

- El archivo `config.rs` está disponible pero actualmente no se usa (fue creado previamente)
- El `build.rs` compila los archivos Vala a C antes de compilar Rust
- La integración D-Bus es crítica: el agente debe registrarse correctamente en el bus
